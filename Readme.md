# Clipboard Center
<img width="512" height="512" alt="512px" src="https://github.com/user-attachments/assets/a6d90b31-9807-4ae2-aec5-63d743e9d632" />

Clipboard Center is a sleek, modern clipboard manager for macOS, built completely with SwiftUI and AppKit. It seamlessly keeps track of your copied text and images, storing them securely for quick retrieval whenever you need them.
## ✨ Features
- **Persistent History**: Automatically saves all your copied text and images securely using CoreData.
- **Quick Access**: Access your clipboard history instantly from the menu bar via a popover, or run it as a standalone window in your dock.
- **Global Hotkey**: Instantly summon the clipboard manager using a customizable global hotkey (default is `Cmd + Shift + V`).
- **Search Capabilities**: Quickly find previously copied items using the built-in search bar.
- **Customizable UI**: Choose between a Menu Bar Extra (Accessory mode) or a regular Dock app based on your workflow.
## 🛠️ Technical Stack
- **Platform**: macOS (supported macOS 13 Ventura and above)
- **UI Framework**: SwiftUI & AppKit
- **Storage**: CoreData for persistent and efficient local storage.
- **Language**: Swift
## Getting Started
1. Open `ClipboardCenter.xcodeproj` in Xcode.
2. Build and run the project (`Cmd + R`).
3. By default, Clipboard Center will appear in your macOS menu bar. 
4. Click the menu bar icon or press `Cmd + Shift + V` to view your clipboard history. Right-click the menu bar icon to access Settings or quit the app.
## Settings
You can customize Clipboard Center by opening the Settings menu:
- **Icon Mode**: Switch between Menu Bar and Dock mode.
- **Hotkey**: Change the global keyboard shortcut to summon the app.
